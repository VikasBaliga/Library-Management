# Library Book Borrowing System

A simple beginner-friendly **Library Management System** built using **Python and Flask**.  
The application allows users to view books, borrow and return them with membership limits, track due dates, and calculate fines for late returns.

---

## Features

- List of available books with copies
- Borrow books
- Return borrowed books
- Due date tracking of 14 days
- Fine calculation for late returns (â‚¹10 per day)
- Add new books to the library

---

## Tech Stack

- Python
- Flask
- SQLite
- HTML (with inline CSS)

---

## Setup Instructions

1. Clone the repo
2. Run using command: python app.py

## Sample Flow
1. Open the application to see the list of available books
2. Add a new book using Add Book
3. Borrow a book
4. Try borrowing more than allowed makes the system block it
5. Return a book
6. If returned after due date then the fine is calculated automatically
